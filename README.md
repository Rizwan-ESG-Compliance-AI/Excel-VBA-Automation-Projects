# Excel VBA Automation Projects

This repository contains Excel VBA automation projects focused on compliance, reporting, data cleaning, dashboards, and workflow improvement.

## Project 1: Solvent Batch Certificate PDF Automation

This Excel VBA macro automates the saving and printing of solvent or chemical batch certificate sheets.

The macro allows users to save the active worksheet as a PDF into a selected workbook-specific folder. It includes a pre-save checklist, automatic file naming using product name, certificate date, and batch number, sheet validation, print setup, and one-click printing.

This type of automation can support chemical, solvent, laboratory, quality control, and compliance documentation workflows.

## Key Features

- One-click PDF saving from the active Excel sheet
- Pre-save checklist before saving or printing
- Automatic file naming using product name, certificate date, and batch number
- Workbook-specific folder memory
- Option to change the PDF saving folder
- Validation for red warning cells
- Validation for empty green required cells
- A4 print setup with automatic page orientation
- One-click printing
- Button creation directly inside Excel
